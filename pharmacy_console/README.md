### fabric 相关操作

正式确认审核通过前, 需为成员:

  1. 注册 CA 用户
  2. 注册 fabric client 用户并配置证书, 将 client 用户信息更新至联盟成员的注册信息中, 用以在获取联盟成员的节点信息和调用合约时使用
  3. 注册 peer 并配置证书, 启动 peer
  4. 获取节点列表和信息时, 直接调用 fabric-manage 相应接口
